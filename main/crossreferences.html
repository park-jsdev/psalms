Minimum Requirements:

HTML5

A. Appropriate HTML5 tags, including use of HTML5 structural tags.

B. Inline images both static and clickable. The logo is a static image. The animation below is clickable.

C. Use of appropriate special characters. See below for & usage.

D. Use of appropriate <meta> tags. See the meta tags, used for viewport for responsive design and description for SEO optimization.

E. A footer on the home page indicating when the website was last updated or modified. The footer also includes a legal disclaimer for this business's purposes.

CSS

A. Use CSS3 to format the appearance of paragraphs, lists, headers, background images, tables, and forms. Styling was used. See "main.css".

B. Use CSS3 contextual selectors (e1, e2, e3; e f; e > f; e + f) and at least two attribute selectors, excluding class and id attribute selectors. Contextual selectors and attribute selectors were used in the footer, in the navbar, and in responsive design. See "main.css".

C. Use of CSS3 styles for color(RGBA & HSLA). Both RGBA and HSLA were used, see "forms.css".

D. Use of dynamic pseudo-classes to highlight hyperlinks states of link, visited, active, hover; focus and two structural pseudo-classes: first-of-type and last-of-type. Dynamic pseudo-classes were used in the navbar.

E. Use of CSS3 background styles (background size, background-clip, background-origin, padding, margins, borders, use of an appropriate border style, use of rounded corners). Background styles were used in the creation of the rows.

F. Your home page should have a drop cap created with CSS3. Drop caps were made using spans and CSS.

G. Define document content with the class and id properties and create styles for them. Class and id properties were used for many elements.

H. Use each of the following page layout techniques: i. Grid ii. Flex iii. Float iv. Positioning - Flex was used for responsive design in the rows. Float and Positioning were used for positoning of content in rows.

Web Form

A. Use of appropriate form elements including some of the new HTML5 form elements. A multistep web form was used.

B. Apply appropriate styles to web forms. Styling was used, in a separate stylesheet than the main page, with the addition of more forms in mind. See "forms.css".

Multimedia

A. Use sound and video somewhere on your site. Ensure these are of short duration, preferably about 30 seconds. A video was used to create a scrolling video animation background for one of the rows. It is also clickable to produce a tab containing more information. In production, a shorter sample of the video itself could be used for improved performance. The script was based on: Reddit thread and in a future version or project, I hope to produce an effect such as: Codepen link.

B. Use of transitions and animations. See above.

JavaScript

A. Use of variables, arithmetic, comparison and logical operators, subroutines, functions, arrays, conditional statements, conditional operators, and running time commands with system clock. JS functions were used to not only create the scrolling video effect, but to have each row expand a corresponding tab and scroll to it, containing more info.

C. Validating form input by using appropriate JavaScript functions. This multi-step form input validation through the validateForm() function in "functions.js" and only allows to pass to the next step if input is valid.

D. Use of JavaScript alerts for validation purposes. If the input validation returns false, will display a window alert, handled in the validateForm() function.

Responsive Design

A. Your website design should include responsive design implementation using concepts covered in this course that would accommodate desktop, tablet, and mobile designs, including positioning, grid, and flex layouts. The website mainly incorporates an absolute layout inside a relative layout to address responsive design. It also uses flex for smaller screen sizes.

B. The use of contextual selectors to control sub level menus is required, including a navicon image for mobile design. Contextual selectors were used to design this animation background with overlay text.

C. Images should also be responsive for different screen sizes and devices. The logo image and animation are responsive.
